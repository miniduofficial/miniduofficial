Sample Space Compression Using a Random Variable

Consider the random experiment of tossing two fair coins.

The sample space is given by

\[ \Omega=\{HH,HT,TH,TT\}. \]

Each element in $\Omega$ is a complete possible outcome, where by complete we mean that each outcome describes the physical phenomenon under consideration with enough detail as required.

Now we define

X = number of heads observed.

We can hence observe that $X$ acts as a function where the domain is the set of all possible outcomes and the codomain is $\mathbb{R}$.

Sample space Ω

HH HT TH TT
X → → → →

Values in ℝ

2 1 1 0

A random variable maps complete outcomes to numerical values.

\[ X:\Omega\to\mathbb{R}. \]

More explicitly,

\[ X(HH)=2,\qquad X(HT)=1,\qquad X(TH)=1,\qquad X(TT)=0. \]

We can see that $X$ converts the outcomes in the sample space into numbers.

The name random variable does not come from the function itself, but rather from the uncertainty of not knowing which $\omega\in\Omega$ will occur. Once the outcome is known, the value of $X$ is determined.
