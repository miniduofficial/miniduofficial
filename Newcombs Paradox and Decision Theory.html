Newcomb's Paradox: An Attack on Free Will?

The Setup

Newcomb's paradox is a thought experiment in decision theory. There are two boxes. Box A is transparent and visibly contains $1,000. Box B is opaque and contains either $1,000,000 or nothing.

The agent has two possible choices. They may take only Box B, or they may take both Box A and Box B. There is also a predictor called Omega. Omega is stipulated to be highly accurate at predicting what the agent will choose.

Omega follows a simple rule. If Omega predicts that the agent will take only Box B, Omega places $1,000,000 in Box B. If Omega predicts that the agent will take both boxes, Omega leaves Box B empty.

By the time the agent makes the choice, Omega has already made the prediction and Box B has already been filled or left empty. The question, then, is deceptively simple: should the agent take only Box B, or take both boxes?

At first glance, the problem seems simple. Taking both boxes gives the agent Box A in addition to whatever is already in Box B. But the complication is that the agent's choice is strongly correlated with Omega's earlier prediction.

This is why Newcomb's paradox creates a disagreement between two approaches to decision theory: Evidential Decision Theory, which recommends one-boxing, and Causal Decision Theory, which recommends two-boxing. To understand why, we first need to distinguish conditional probability from causal or interventional probability.

Conditional Probability

Conditional probability asks: if I learn that event B is true, how should I update my probability that event A is true?

Suppose we have a sample space \(\Omega\), and two events \(A\) and \(B\) inside that sample space. If we learn that \(B\) is true, then the possibilities outside \(B\) are no longer relevant to the question being asked. The sample space is restricted to \(B\).

Inside this restricted sample space, event \(A\) occurs only in those possibilities where both \(A\) and \(B\) occur. That is, the relevant region is: \[ A \cap B. \]

Therefore, provided that \(P(B) \neq 0\), conditional probability is defined as:

\[ P(A \mid B) = \frac{P(A \cap B)}{P(B)}. \]

Intuitively:

\[ P(A \mid B) \]

means: Among the worlds where B is true, how often is A also true?

This does not mean that \(B\) causes \(A\). It only means that, once \(B\) is known, \(A\) becomes more or less probable relative to that restricted space of possibilities.

For example, suppose we observe many people wearing raincoats. Then:

\[ \Pr(\text{Rain} \mid \text{Raincoats}) \]

may be high. But this does not mean that wearing raincoats causes rain. Rather, raincoats are evidence for rain. They indicate rain, but they do not produce rain.

So conditional probability tracks evidential relevance, not necessarily causal influence.

Interventional Probability

Interventional probability asks a different question. Instead of asking what should I believe if B is true, it asks what would happen if I made B true?

P(A | do(B))

This should be read as: the probability of A if B were actively done or intervened upon.

The difference between conditional and interventional probability is crucial. P(A | B) asks what the probability of A is given that B is observed. P(A | do(B)) asks what the probability of A is if B is forcibly made true.

The raincoat example makes this distinction clear. If I observe raincoats, then rain becomes more probable. But if I force people to wear raincoats on a dry day, that does not cause rain. Conditional probability is about learning from evidence. Interventional probability is about the causal consequences of making something happen.

Decision Theory Fundamentals

Decision theory studies how an agent should choose among possible actions when those actions may lead to different outcomes under uncertainty.

A basic decision problem contains an agent, a set of possible actions, a set of possible states of the world, a set of possible outcomes, probabilities over uncertain events, and a utility function representing the desirability of outcomes.

An agent is the entity whose decision is being evaluated. The agent may be a human being, an animal, an institution, or an artificial system. An action is one of the choices available to the agent. A state of the world is a relevant fact or condition that affects what outcome follows from an action. An outcome is what results from an action together with a state of the world.

A utility function assigns a numerical value to each possible outcome. If Oᵢ is an outcome, then U(Oᵢ) represents the utility of that outcome.

Once we have actions, outcomes, probabilities, and utilities, we can define expected utility. The expected utility of an action is the average utility the agent can expect from that action, once each possible outcome is weighted by its probability.

EU(A) = Σᵢ P(Oᵢ | A)U(Oᵢ)

This says: for action A, multiply each possible outcome's utility by its probability, and add the results. The action with the highest expected utility is then treated as the rational action, at least under standard expected utility theory.

However, Newcomb's paradox creates a problem because different decision theories disagree about which probability should be used in the expected utility calculation.

Evidential Decision Theory and Causal Decision Theory

Evidential Decision Theory, or EDT, evaluates an action by asking what that action would be evidence for.

It asks: if I choose action A, what does that tell me about which outcome is likely?

EEU(A) = Σᵢ P(Oᵢ | A)U(Oᵢ)

Here, P(Oᵢ | A) means the probability of outcome Oᵢ given that the agent chooses action A. So EDT treats the action as evidence. The action may reveal something about the world, even if it does not causally produce that thing.

Causal Decision Theory, or CDT, evaluates an action differently. It asks: if I were to perform action A, what would that action causally bring about?

CEU(A) = Σᵢ P(Oᵢ | do(A))U(Oᵢ)

So CDT treats the action not primarily as evidence, but as an intervention.

The contrast is therefore simple but deep. EDT asks: what does my action indicate? CDT asks: what does my action causally affect?

In ordinary life, these often coincide. Taking medicine may be evidence that one will recover because the medicine causally contributes to recovery. But Newcomb's paradox separates evidence and causation. This is why the paradox is philosophically powerful.

Newcomb's Paradox Revisited

Let A₁ be one-boxing and A₂ be two-boxing. There are two relevant states of the world: Box B is full, and Box B is empty.

Box B Full Box B Empty
One-box $1,000,000 $0
Two-box $1,001,000 $1,000

At first glance, two-boxing seems to dominate one-boxing. If Box B is full, then two-boxing gives $1,001,000, whereas one-boxing gives $1,000,000. So two-boxing gives $1,000 more.

If Box B is empty, then two-boxing gives $1,000, whereas one-boxing gives $0. Again, two-boxing gives $1,000 more.

So, if the contents of Box B are treated as already fixed, two-boxing seems better in every possible case. This is the core intuition behind causal decision theory's recommendation.

But there is another fact: the contents of Box B are not random in the ordinary sense. They depend on Omega's prediction, and Omega's prediction is highly correlated with the agent's action.

The Evidential Calculation

Evidential Decision Theory asks what the agent's action is evidence for. If the agent one-boxes, this is evidence that Omega predicted one-boxing. If Omega is p-accurate, then:

P(Box B full | One-box) = p

Similarly:

P(Box B empty | One-box) = 1 − p

Therefore, the evidential expected utility of one-boxing is:

EEU(One-box) = p(1000000) + (1 − p)(0) = 1000000p

Now consider two-boxing. If the agent two-boxes, this is evidence that Omega predicted two-boxing. If Omega predicted two-boxing, Box B is empty.

P(Box B empty | Two-box) = p
P(Box B full | Two-box) = 1 − p

Therefore, the evidential expected utility of two-boxing is:

EEU(Two-box) = (1 − p)(1001000) + p(1000)

Expanding:

EEU(Two-box) = 1001000 − 1000000p

EDT recommends one-boxing when:

1000000p > 1001000 − 1000000p

Hence:

p > 1001000 / 2000000 = 0.5005

This is a striking result. Under these assumptions, EDT recommends one-boxing whenever Omega is more than 50.05% accurate.

So Omega does not need to be perfect. Omega does not even need to be nearly perfect. It only needs to be slightly better than chance.

For example, suppose Omega is 99% accurate. Then p = 0.99. For one-boxing:

EEU(One-box) = 1000000(0.99) = 990000

For two-boxing:

EEU(Two-box) = (0.01)(1001000) + (0.99)(1000) = 11000

Therefore, with a 99% accurate Omega, one-boxing has evidential expected utility $990,000, whereas two-boxing has evidential expected utility $11,000. So EDT strongly recommends one-boxing.

The reason is not that one-boxing causally makes Box B full. Rather, the reason is that one-boxing is strong evidence that Omega predicted one-boxing, and therefore strong evidence that Box B is full.

The Causal Calculation

Causal Decision Theory asks a different question. It does not ask what the action indicates. It asks what the action causally affects.

By the time the agent chooses, Omega has already made its prediction. Box B has already been filled or left empty. Therefore, the causal decision theorist argues that the agent's present action cannot causally affect whether Box B is full.

Let q = P(Box B full) from the agent's current perspective. This q represents the agent's uncertainty about whether Box B is full.

If the agent one-boxes, then:

CEU(One-box) = q(1000000) + (1 − q)(0) = 1000000q

If the agent two-boxes, then if Box B is full, the agent receives $1,001,000. If Box B is empty, the agent receives $1,000.

CEU(Two-box) = q(1001000) + (1 − q)(1000)

Expanding:

CEU(Two-box) = 1000000q + 1000

Therefore:

CEU(Two-box) = CEU(One-box) + 1000

So, whatever value q has, two-boxing gives $1,000 more than one-boxing. If Box B is full, two-boxing gives $1,000 more. If Box B is empty, two-boxing also gives $1,000 more. Therefore, CDT recommends two-boxing.

The reason is not that Omega is unreliable. Rather, the reason is that the contents of Box B are already fixed by the time the agent chooses. The agent's present action can affect whether the agent takes Box A, but it cannot causally affect what Omega already predicted or what Omega already placed in Box B.

The Divergence

The paradox can now be stated clearly. EDT says: choose according to what your action indicates. CDT says: choose according to what your action causally affects.

In Newcomb's paradox, these two come apart. One-boxing is strong evidence that Box B is full, but it does not causally make Box B full. Two-boxing causally gives the agent the visible $1,000, but it is also evidence that Omega predicted two-boxing and left Box B empty.

So the disagreement is not about whether $1,001,000 is greater than $1,000,000. Everyone agrees that it is. Nor is it about whether $1,000 is greater than $0. Everyone agrees that it is.

The disagreement is about which probability belongs in the expected utility calculation.

EDT uses P(Oᵢ | A): what is the probability of outcome Oᵢ given that I choose action A?

CDT uses P(Oᵢ | do(A)): what is the probability of outcome Oᵢ if I intervene and perform action A?

In ordinary cases, these two probabilities often point in the same direction. But Newcomb's paradox constructs a case where they diverge. The paradox therefore exposes a fracture between evidence and causation.

A Deeper Objection: Can CDT Intervene on an Agent's Action?

The usual presentation of Newcomb's paradox asks whether the rational agent should one-box or two-box. But there is a deeper issue.

Causal Decision Theory evaluates an action by asking what would happen if the agent were to perform that action as an intervention:

P(Oᵢ | do(A))

This raises a difficult question: what does it mean to intervene on an agent's action?

In ordinary causal examples, the intervention is relatively easy to understand. P(Rain | do(People wear raincoats)) asks what would happen if we forced people to wear raincoats while leaving the weather system unchanged. The answer is clear: forcing people to wear raincoats does not cause rain.

But when the intervention is performed on an agent's decision, the matter becomes less straightforward.

An agent's decision is not an isolated event floating freely in the world. It is produced by the agent's beliefs, desires, memories, character, reasoning, neural state, environment, and causal history.

So when CDT asks P(Box B full | do(Two-box)), it seems to imagine that the agent's action can be surgically altered while the rest of the relevant causal structure remains fixed.

But this may be an illegitimate idealization. If Omega predicted the agent by modelling the agent's psychology, reasoning, or decision procedure, then the agent's final choice is not independent of the very facts Omega used to make the prediction.

The action is downstream of the agent's internal structure. Omega's prediction is also downstream of that same structure. So the action and Omega's prediction may share a common cause: the agent's decision-making process.

In that case, intervening only on the final action may sever the action from the process that makes it the agent's action.

CDT says that the present act of two-boxing does not causally affect whether Box B is full. This is true if the action is treated as a detachable intervention. But if the agent's action is the result of a prior causal structure that Omega has already modelled, then do(A) may no longer describe the original decision problem.

Instead, it describes a modified world where the action is forced independently of the agent's actual decision-making process.

The Free Will Pressure

The difficulty becomes sharper if we reject libertarian free will.

If agents do not possess a contra-causal capacity to choose independently of prior causes, then an action cannot simply be treated as an uncaused intervention point. The action arises from the agent's causal history.

To change the action, one would usually have to change something upstream, such as the agent's beliefs, preferences, reasoning process, neural state, environment, or prior causal history.

But if those upstream factors are changed, then Omega's prediction may also have been different.

So the CDT intervention do(Two-box) may be too thin. It asks us to imagine the action being changed while holding fixed the causal background that, in the original setup, helped determine both the action and Omega's prediction.

This suggests that CDT may presuppose an idealized model of agency: one in which the agent's action can be treated as an intervention variable, separable from the causal network that produced it.

But Newcomb's paradox pressures precisely this assumption. The paradox is not only about whether one-boxing or two-boxing is rational. It is also about whether an agent's action can be cleanly separated from the causal and evidential structure that makes it predictable.

If the action is treated as a free-floating intervention, CDT recommends two-boxing. If the action is treated as the output of the agent's actual decision procedure, which Omega has already predicted, then one-boxing becomes more compelling.

The Central Philosophical Question

The deeper issue is not merely which action maximizes expected utility. It is also: what kind of thing is an action?

Is an action an intervention that can be surgically inserted into the world? Or is an action the visible output of an agent's entire causal, psychological, and computational structure?

Newcomb's paradox becomes philosophically interesting because CDT seems to require the first picture, while the setup itself seems to push us toward the second.

This is why Newcomb's paradox can be read as an attack on a certain conception of free will. It does not simply show that one-boxing or two-boxing is correct. Rather, it exposes a tension in how we model agency.

If the agent's action is treated as an independent intervention point, then CDT's reasoning appears valid. But if the agent's action is treated as the determined output of a causal process that Omega has already successfully modelled, then CDT's intervention on the action may no longer correspond to the original decision problem.

The paradox therefore asks us to examine a hidden assumption behind causal decision theory:

Can an action be separated from the causal process that produces it?

If the answer is yes, CDT has a natural route to two-boxing. If the answer is no, then the CDT analysis may be idealizing away the very structure that makes Newcomb's paradox possible.

Thus, Newcomb's paradox is not merely a puzzle about two boxes. It is a puzzle about agency, prediction, causation, and the metaphysical status of choice itself.
